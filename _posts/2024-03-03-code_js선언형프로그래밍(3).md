## 표현력을 높여주는 도구들(심화)
### 1. go
값을 하나 받아 함수를 순차적으로 실행시키는 함수
```
go(
    0,
    a => a + 1,
    a => a + 10,
    a => a + 100
    );
// 값을 하나 받아서 순차적으로 함수를 실행
``` 
```javascript
const go = (...args) => reduce((a, f) => f(a), args);

```

### 2. pipe
함수를 순차적으로 실행시키는 함수를 만드는 함수
```
const f = pipe(
    a => a + 1,
    a => a + 10,
    a => a + 100
    ); 
// 순차적으로 함수를 실행하는 f 함수를 만들기
```
```javascript
const pipe = (f, ...fs) => (...as) => go(f(...as), ...fs);
```

### 3. curry
```
f(a, b, c) -> f(a)(b)(c)
```
```javascript
const curry = f =>
  (a, ..._) => _.length ? f(a, ..._) : (..._) => f(a, ..._);
```
